Features

Everything you need,
nothing you don't

Stellarr is designed to be simple yet powerful. No bloat, no complexity — just a clean interface for managing media requests.

Discover & Request

Search TMDB's massive catalog of movies and TV shows. Browse trending content, request with one click, and track everything in a beautiful interface.

Automatic Fulfillment

Native import lists for Sonarr and Radarr mean your requests are automatically picked up and downloaded. No manual intervention required.

Plex Integration

Webhooks automatically track when requested media arrives in your Plex library. Users get push notifications when their shows and movies are ready.

Progressive Web App

Install on any device. Works beautifully on phones, tablets, and desktops with native app-like experience.

Secure by Default

Client-side key derivation with PBKDF2. Your password never leaves the browser. HMAC-signed session tokens.

Multilingual

Built-in support for English, Spanish, French, and German with automatic browser language detection.

RSS Feeds

Native RSS and JSON import list endpoints. Works out of the box with Sonarr and Radarr list imports.

Lightweight

Static SvelteKit frontend with a lean Python backend. No Node.js runtime needed in production. Tiny resource footprint.

Push Notifications

Get notified instantly when your requested media is added to the library via web push notifications.

your screen in minutes

Search & Request

Users search TMDB's catalog and request movies or TV shows with a single click. Browse trending content or search for anything.

Deployment

Deploy your way

Choose the deployment that fits your setup. Both options give you the full Stellarr experience.

Docker

Self-Hosted

Run on your own hardware with Docker Compose. SQLite database, Caddy reverse proxy with automatic HTTPS. Full control, zero cloud costs.

docker compose up -d
  • One-command setup
  • Auto HTTPS with Caddy
  • SQLite — no database server
  • Zero ongoing costs

AWS Serverless

Cloud

Deploy to AWS Lambda with DynamoDB, CloudFront CDN, and S3. Scales automatically, costs under $1/month. Infrastructure as code with Terraform.

terraform apply
  • Auto-scaling, zero maintenance
  • CloudFront CDN globally
  • DynamoDB on-demand pricing
  • ~$0.50/month typical cost
